# DotDev Theme for Visual Studio Code

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/dotdevdotdev/dotdev-theme)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE.md)
[![VS Code](https://img.shields.io/badge/VS%20Code-^1.53.2-blue.svg)](https://code.visualstudio.com/)

DotDev is a dark theme for Visual Studio Code designed for developer productivity.

## 🎨 Theme Features

- **DotDev** - A dark theme with vibrant neon accents designed for long coding sessions
- Carefully selected colors for improved code readability
- Reduced eye strain with balanced contrast ratios
- Distinct syntax highlighting for different token types

## 📦 Installation

### Via Visual Studio Code Marketplace

1. Open **Extensions** sidebar panel in VS Code. `View → Extensions`
2. Search for `DotDev Theme`
3. Click **Install**
4. Click **Reload** to reload your editor
5. Navigate to `Code > Preferences > Color Theme` and select your preferred DotDev theme

### Via Command Line

```bash
code --install-extension dotdevdotdev.dotdev-theme
```

## 🚀 Usage

Once installed, you can activate the DotDev theme:

1. Open the Command Palette (`Cmd/Ctrl + Shift + P`)
2. Type "Color Theme" and select `Preferences: Color Theme`
3. Choose "DotDev" from the list
