# Change Log

All notable changes to the "DotDev Theme" extension will be documented in this file.

## [1.0.0] - 2025-09-11

### Added

- Initial release of DotDev Theme
- Theme variations:
  - DotDev (main dark theme)
- Support for popular programming languages and frameworks
- Optimized color schemes for better readability and reduced eye strain
- Custom workbench colors for a cohesive coding experience

### Features

- Syntax highlighting for JavaScript, TypeScript, Python, PHP, HTML, CSS, SCSS
- Framework-specific optimizations for Vue.js, React, Django
- Carefully selected color palette for improved code readability
- Distinct colors for different token types
- Enhanced visibility for active selections and highlights
